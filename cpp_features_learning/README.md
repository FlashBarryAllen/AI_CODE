# C++ Features Learning Project

This project demonstrates various C++ features through practical examples and test cases. It covers modern C++ concepts including C++11, C++14, and C++17 features.

## Features Covered

1. **Auto keyword**: Automatic type deduction
2. **Range-based for loops**: Modern iteration syntax
3. **Smart Pointers**: `std::unique_ptr`, `std::shared_ptr`, and `std::weak_ptr`
4. **Lambda Expressions**: Anonymous functions
5. **STL Containers**: Vectors, maps, sets with algorithms
6. **RAII (Resource Acquisition Is Initialization)**: Automatic resource management
7. **Move Semantics**: Move constructors and move assignment operators
8. **Threading**: Multithreading with mutexes
9. **Async Programming**: Asynchronous function execution

## How to Build and Run

```bash
mkdir build
cd build
cmake ..
make
./main
```

## Test Cases Included

The program includes comprehensive test cases for:

- Basic C++11/14/17 features
- STL containers and algorithms
- RAII and move semantics
- Threading and synchronization
- Asynchronous programming

Each test case outputs its results to demonstrate how the feature works in practice.
